# Story

Story is website using react and firebase allow user to take photo as story and share it in 24 houres so other user can see it.

## Technology used

- React.
- Context API.
- Css Modules.
- Material UI.
- react-webcam.
- react-timeago
- Firebase Auth.
- Firebase Firestore.
- Firebase Storage.
